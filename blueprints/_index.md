# Blueprint Index

Blueprints are complete aesthetic directions. Each defines a mood, visual logic, palette family recommendation, component vocabulary, and usage constraints.

Choose the blueprint that fits the product's category, audience, and emotional register. When no single blueprint fits cleanly, blend — but name the blend explicitly and identify the primary direction.

See `decisions/blueprint-selection.md` for a full selection decision framework.

---

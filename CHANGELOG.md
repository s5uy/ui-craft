## Versioning Convention

`v[major].[minor].[patch]`

- **Major** — architectural changes, SKILL.md restructure, breaking changes to how the skill works
- **Minor** — new blueprints, new palette families, new component pattern subdirectories
- **Patch** — additions to existing files, new component patterns within existing subdirectories, new prompt templates, typo fixes

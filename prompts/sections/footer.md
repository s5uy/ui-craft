# Section Prompt: Footer

---

## Generation Scaffold

```
Generate a footer for [product name].

**Design system:**
- Blueprint: [chosen]
- Palette: [chosen]

**Footer complexity:** [minimal / standard / full]

Minimal: logo + copyright + 2–3 links (privacy, terms)
Standard: logo + description + 2–3 link columns + copyright
Full: logo + description + 4–5 link columns + social icons + copyright

**Link columns (for standard/full):**
Column 1: [name, links list]
Column 2: [name, links list]
Column 3: [name, links list]

**Social links (for full):** [Twitter/X, GitHub, LinkedIn, etc.]
**Legal:** Copyright year, company name, privacy link, terms link

**Anti-patterns:**
- Footer should not repeat the hero CTA — the user made their decision; don't re-sell
- Link columns should only contain links the user might actually need from the footer
- No gradient background on footer

**Output the complete footer component.**
```
